# Null Bugs - Digital Solutions Website

A professional, responsive multi-page website for **Null Bugs**, a digital agency specializing in web development, mobile apps, and UI/UX design.

## Approach & Methodology
The goal was to create a modern, high-conversion landing experience for a tech-focused service provider. The design follows a "Clean & Dark" aesthetic with high-contrast gradients to emphasize digital innovation.

### Key Features:
* **Modular Design:** Used a consistent grid system (2, 3, and 4-column layouts) to ensure content is digestible.
* **Responsive Architecture:** Implemented a mobile-first approach with a toggleable navigation menu for smaller screens.
* **Interactive Elements:** Included background "blobs" and fade-up animations (via CSS/JS) to enhance user engagement.
* **Direct Conversion:** Integrated WhatsApp "click-to-chat" floating buttons and CTA links to reduce friction in the sales funnel.

## Tools & Technologies
* **HTML5 & CSS3:** Semantic markup and custom CSS variables for consistent branding (colors, spacing, and radius).
* **Boxicons:** For high-quality, scalable vector icons.
* **JavaScript:** Custom logic for mobile menu toggling and current year auto-updates in the footer.
* **Google Fonts & Unsplash:** For professional typography and high-resolution imagery.

## Notable Design Decisions
* **Gradient Accents:** Used a text-gradient (primary to accent) on headings to draw the eye to key value propositions.
* **Glassmorphism:** Applied subtle surface transparency and borders to cards to create depth against the dark background.
* **Social Proof & Trust:** Strategically placed a "Development Process" section to build transparency and trust with potential clients.
